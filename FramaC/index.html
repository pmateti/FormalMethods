1 What is Frama-C?

  1. http://frama-c.com Frama-C is a suite of tools dedicated to the compile-time analysis of the source code written in C/C++. Assertions are written by the programs. Frama-C can then generate math logic formaulae whose proofs imply the correctness of programs. Proofs are carried out by mechanized theorem provers.
    1. Installing Frama-C; https://opam.ocaml.org/; http://opam.ocaml.org/doc/Install.html;
    2. QA https://stackoverflow.com/tags/frama-c/;

2 ACSL

  1. ANSI/ISO C Specification Language

2.1 A Quick Example

/*@ requires \valid(a) && \valid(b);
  @ ensures A: *a == \old(*b) ;
  @ ensures B: *b == \old(*a) ;
  @ assigns *a,*b ;
  @*/
void swap(int *a,int *b)
{
  int tmp = *a ;
  *a = *b ;
  *b = tmp ;
}
  1. requires \valid(a) && \valid(b); entry assertion (also called pre condition)
  2. ensures A: *a = \old(*b) ; exit assertion (also called post condition)
  3. assigns x; The value of x at exit may be different from that at entry.
  4. \valid(x) builtin predicate: The value x is valid
  5. \old(x) builtin : The value x had at entry
  1. Alexey Khoroshilov Mikhail Mandrykin, Specifying and Proving Correctness of Linux Kernel Components with ACSL, 35+ slides, 2017, Institute for System Programming of the Russian Academy of Sciences; {pm: Has several Frama-C examples}
