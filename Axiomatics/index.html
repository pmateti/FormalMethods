2 What is an Axiom?

  1. An axiom is an expression in mathematical logic.
  2. We always consider a collection of axioms.
  3. Truthful?
  4. Any other expectations?

2.1 Expectations

  1. A collection of axioms describe worlds of our interest
    1. A world where all the axioms are [held to be] true.
    2. There can be worlds where one or more of the axioms are [held to be] false. (We are not interested in these.)
    3. Must there be a world where all the axioms are true?
  2. Axioms in a collection ought to be independent. [No axiom is deducible from others.]
  3. The collection is expected to be not contradictory. [sound]
  4. The collection may not be "complete". [truth]

2.2 Dedekind-Peano Axioms [ 1889] for Numbers

  1. The 0 is a natural number.
  2. For every natural number n, S(n) is a natural number. [S is a function.]
  3. For all natural numbers m and n, m = n iff if S(m) = S(n).
  4. For every natural number n, S(n) = 0 is false.

2.3 Equality Relation

  1. The following are valid in First-Order Logic with Equality.
  2. For every natural number x, x = x. [reflexive]
  3. For all natural numbers x and y, if x = y, then y = x. [symmetric]
  4. For all natural numbers x, y and z, if x = y and y = z, then x = z. [transitive]
  5. For all a and b, if b is a natural number and a = b, then a is also a natural number. [Natural numbers are closed under equality.]

3 Semantics of Assignment by an Axiom

  1. Consider x := e, where x is a variable, and e an expression.
  2. Assume that x is a whole variable, not, e.g., part of an array as in a[i].
  3. Assume that the value e computes is well-defined and of the same type as x.

3.1 {Q(e)} x := e {Q(x)}

3.2 Floyd-Hoare Triples {P} S {Q}

  1. S is code in some PL; P and Q are assertions.
  2. Th entire triplet yields true/false.
  3. {P} S {Q} is "operationally defined" as: For all states s that satisfy P, executing S on s, the resulting state s' is such that s' satisfies Q, if and when S terminates.
  4. Note that the above is about partial correctness: It omits termination.
